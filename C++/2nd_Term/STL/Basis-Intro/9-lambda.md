lambda表达式：核心————简化代码书写，构建临时需要调用的函数
只使用一次的函数对象，不用专门为其编写一个类；只调用一次的简单函数，可以在调用时才写出其函数体。

基本形式：[外部变量访问方式说明符](参数表) -> 返回值类型 { 语句组 }
1）关于外部变量访问方式说明符：[]里面的内容
[] 	        不使用任何外部变量
[=] 	    以传值的形式使用所有外部变量
[&]	        以引用形式使用所有外部变量
[x, &y] 	x 以传值形式使用，y 以引用形式使用
[=,&x,&y]	x,y 以引用形式使用，其余变量以传值形式使用
[&,x,y] 	x,y 以传值的形式使用，其余变量以引用形式使用
2）关于返回值类型：“->返回值类型”也可以没有， 没有则编译器自动判断返回值类型。 

一些使用的样例：
[ ](double a,double b) { return a + b; }(1.2,2.5)
**后面的 (1.2, 2.5) 是调用这个 lambda 表达式的参数列表。**
(1.2, 2.5) 就是调用运算符，没有它，lambda 只是定义，不会执行。而有了它就会传入a = 1.2和b = 2.5并调用函数执行，返回3.7这个值

[=,&y,&z](int n) { cout << x << endl; y++; z++; return n*n; };

int a[4] = { 4,2,11,33};
sort(a,a+4,[ ](int x,int y)->bool { return x%10 < y%10; });
for_each(a,a+4,[ ](int x) {cout << x << " " ;} ) ;

for_each(a.begin(),a.end(),[&](int & x) {total += x; x*=2;});
for_each(a.begin(),a.end(),[ ](int x) { cout << x << " ";});

function<int(int)> fib = [&fib](int n) { return n <= 2 ? 1 : fib(n-1) + fib(n-2);};
**function<int(int)> 表示返回值为 int, 有一个int参数的函数**
lambda使用这种function<int(int)>标识给一个函数赋值才能达成递归调用，直接自己来无法进行递归
